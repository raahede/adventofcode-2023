# adventofcode-2023

Setup
```bash
yarn
```

Solve a puzzle
```bash
# Dec 1, part A
npx tsx src/1/a

# Dec 2, part B
npx tsx src/2/b
```

Run/watch unit tests

```bash
yarn test
```

Run tests on specific path
```bash
yarn test src/2
```

Run tests on specific function
```bash
yarn test -t getNumbersFromString
```
